# Gemini CLI Devcontainer Feature

This devcontainer feature installs the Gemini CLI and its dependencies.

## Example Usage

To use this feature, add the following to your `devcontainer.json` file:

```json
"features": {
    "ghcr.io/marblejenka/devcontainer-feature/gemini-cli:0": {}
}
```

## Options

| Option | Description | Type | Default |
|---|---|---|---|
| `version` | The version of the Gemini CLI to install. | string | `latest` |

## Customizations

This feature includes the following VS Code customizations:

*   **Extension:** `google.geminicodeassist`
*   **Setting:** `geminicodeassist.updateChannel` is set to `Insiders`.

## Dependencies

This feature depends on the following:

*   [Node.js](https://ghcr.io/devcontainers/features/node:1)
